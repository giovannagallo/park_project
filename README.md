#  Park Project

O **Park Project** é um sistema simples de Estacionamento Automático com interface gráfica.  
Ele simula o funcionamento de um estacionamento real, controlando entrada, saída e quantidade de vagas disponíveis.

---

##  Funcionalidades

- Interface gráfica simples e intuitiva  
- Controle de entrada de veículos  
- Controle de saída de veículos  
- Exibição do número de vagas disponíveis  
- Bloqueio quando o estacionamento está cheio  
- Mensagens visuais de aviso e confirmação  

---

##  Interface

O sistema utiliza uma interface gráfica para facilitar a interação do usuário.  
As ações principais (entrada e saída) podem ser executadas por botões na tela.
